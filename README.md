# geoIACS physical models

The purpose of this repository is to manage the changes to the geoIACS physical models.

The geoIACS data maodel is currently available in the following formats:

* [GeoPackage template](gpkg-template/geoIACS_template.gpkg)
* [GML Application schema](application-schema/geoiacs.xsd)


N.B.: To access the GML application schema from your ETL software tools (such as hale studio, FME, etc.), please use [this link](https://geoiacs.github.io/physical-models/application-schema/geoiacs.xsd).
